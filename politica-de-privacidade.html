Política de Privacidade

Consultor de Campanhas

Última atualização: 08 de Abril de 2026

1. Introdução

A presente Política de Privacidade descreve como o aplicativo Consultor de Campanhas coleta, utiliza, armazena e protege as informações dos usuários.

Nosso compromisso é garantir transparência, segurança e conformidade com as boas práticas de proteção de dados, incluindo a Lei Geral de Proteção de Dados (LGPD).

Ao utilizar o aplicativo, você concorda com os termos desta Política.

2. Sobre o Aplicativo

O Consultor de Campanhas é uma ferramenta de análise e inteligência de marketing, desenvolvida para:

  • Monitorar anúncios públicos disponíveis na internet
  • Analisar dados de campanhas para fins estratégicos
  • Apoiar decisões de marketing e publicidade
  • Gerar insights com base em informações públicas

O aplicativo não cria, edita ou publica campanhas automaticamente.

3. Dados Coletados

O aplicativo pode coletar os seguintes tipos de dados:

3.1 Dados fornecidos pelo usuário

  • Nome
  • E-mail
  • Informações de login (quando aplicável)

3.2 Dados coletados automaticamente

  • Endereço IP
  • Tipo de dispositivo
  • Navegador utilizado
  • Dados de uso da aplicação

3.3 Dados de plataformas de terceiros (Meta/Facebook)

Quando autorizado pelo usuário, o aplicativo pode acessar:

  • Informações públicas de páginas
  • Dados de anúncios públicos
  • Dados de campanhas disponíveis via API oficial da Meta

Importante: O aplicativo acessa apenas dados permitidos pelas permissões concedidas pelo usuário.

4. Como Utilizamos os Dados

Os dados coletados são utilizados para:

  • Fornecer funcionalidades do aplicativo
  • Analisar campanhas e anúncios públicos
  • Gerar relatórios e insights estratégicos
  • Melhorar a experiência do usuário
  • Garantir segurança e prevenção de fraudes

5. Compartilhamento de Dados

Nós não vendemos, alugamos ou comercializamos dados pessoais.

Os dados podem ser compartilhados apenas nas seguintes situações:

  • Com provedores de infraestrutura (ex: servidores, banco de dados)
  • Para cumprimento de obrigações legais
  • Quando exigido por autoridades competentes

6. Armazenamento e Segurança

Adotamos medidas técnicas e organizacionais para proteger os dados, incluindo:

  • Criptografia
  • Controle de acesso
  • Monitoramento de segurança
  • Backups periódicos

Os dados são armazenados apenas pelo tempo necessário para cumprir as finalidades descritas nesta política.

7. Direitos do Usuário

Nos termos da LGPD, você tem o direito de:

  • Acessar seus dados
  • Corrigir informações incompletas ou incorretas
  • Solicitar exclusão dos dados
  • Revogar consentimento
  • Solicitar portabilidade (quando aplicável)

Para exercer seus direitos, entre em contato pelo e-mail abaixo.

8. Cookies e Tecnologias de Rastreamento

O aplicativo pode utilizar cookies e tecnologias similares para:

  • Melhorar a navegação
  • Analisar comportamento de uso
  • Personalizar experiência

Você pode desativar cookies nas configurações do seu navegador.

9. Integrações com Terceiros

O aplicativo pode se integrar a serviços como:

  • Meta (Facebook / Instagram)
  • APIs de marketing e análise

Essas plataformas possuem suas próprias políticas de privacidade, e recomendamos sua leitura.

10. Alterações nesta Política

Esta Política pode ser atualizada a qualquer momento.

Recomendamos que você revise periodicamente esta página.

  • mail E-mail: contato@northwaycompany.com.br

12. Base Legal

O tratamento de dados é realizado com base em:

  • Consentimento do usuário
  • Execução de contrato
  • Cumprimento de obrigações legais
  • Legítimo interesse

13. Observações Importantes

  • O aplicativo trabalha exclusivamente com dados públicos ou autorizados
  • Não realiza coleta indevida de informações privadas
  • Atua em conformidade com as diretrizes da plataforma Meta
